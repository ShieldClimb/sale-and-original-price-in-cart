Show Sale and Original Price in Cart for WooCommerce. Display the original price with a strikethrough, improve price clarity, and boost conversions!

== Description ==

Enhance your WooCommerce store’s pricing visibility with WooCommerce Cart Price. 

Show Sale and Original Price in Cart for WooCommerce effortlessly! 

This plugin ensures that when a product is on sale, the original price appears with a strikethrough, making discounts clear to customers.

See more via [this website](https://shieldclimb.com/free-woocommerce-plugins/sale-and-original-price-in-cart/).

== Installation ==

1. Download our plugin from https://shieldclimb.com/free-woocommerce-plugins/sale-and-original-price-in-cart/
2. At your WordPress admin, Go to Plugins > Add New Plugin > Upload Plugin > Choose File (Choose the file you downloaded just now) > Click Install
3. Click Activate
